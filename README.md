# Task_3_elevatelabs

# About the Project
-> This project is based on creating a responsive layout using CSS Flexbox. The main aim is to practice alignment, spacing, and making the page look good on different screen sizes.

# Tools Used
-> VS Code
->Browser Developer Tools

# What I Used
->External CSS file
->Flexbox (display: flex)
->justify-content and align-items for alignment
->flex-wrap for responsive behavior
->Simple navigation bar and content cards

# Result
->Layout adjusts well on mobile and desktop
->Content looks clean and properly aligned
->Page is easy to read and responsive
